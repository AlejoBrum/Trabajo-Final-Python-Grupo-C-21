## Descripción

Este trabajo consiste en un simulador de cajero automático desarrollado en Python. El programa permite que un usuario ingrese con su DNI y PIN para acceder a un menú con distintas operaciones bancarias.

Las funciones que tiene el sistema son:

- Iniciar sesión.
- Consultar saldo.
- Extraer dinero.
- Depositar dinero.
- Transferir dinero a otro usuario.
- Consultar la cantidad de operaciones realizadas.
- Mostrar mensajes de error cuando se ingresan datos incorrectos.

Para desarrollar el programa utilizamos funciones, estructuras `if` y `while`, validaciones y un diccionario para guardar la información de los usuarios.

## Instrucciones para ejecutar el programa

1. Descargar o clonar el repositorio.
2. Abrir el archivo `Escenario_4.py`.
3. Ejecutar el programa con Python.
4. Ingresar un DNI y un PIN válidos.
5. Elegir la operación que se desea realizar desde el menú.
